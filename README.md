# CloudMzansi

A modern, responsive React web application built with Vite, featuring a clean and professional design for cloud services and digital solutions.

## 🚀 Features

- **Modern React 19** - Built with the latest React features and hooks
- **Vite Build System** - Fast development and optimised production builds
- **React Router DOM** - Client-side routing with smooth page transitions
- **Responsive Design** - Mobile-first approach with responsive breakpoints
- **Component Library** - Reusable UI components with consistent styling
- **Performance Optimised** - Lazy loading and efficient bundle splitting
- **SEO Friendly** - Proper meta tags and semantic HTML structure

## 📁 Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── layout/         # Layout components (Header, Footer)
│   └── ui/            # UI components (Button, Card, etc.)
├── pages/             # Page components
│   ├── Home.jsx       # Landing page
│   ├── About.jsx      # About page
│   ├── Services.jsx   # Services page
│   ├── Work.jsx       # Portfolio/Work page
│   ├── Contact.jsx    # Contact page
│   └── NotFound.jsx   # 404 error page
├── hooks/             # Custom React hooks
├── utils/             # Utility functions and constants
├── data/              # Static data and content
├── assets/            # Images, icons, and static assets
├── App.jsx            # Main application component
├── main.jsx           # Application entry point
└── index.css          # Global styles
```
